# 🛸 우주 미로 (Cosmic Maze)

> Pac-Man 을 닮은 **우주 로그라이크 미로 게임**. 유령이 떠도는 우주 정거장에서
> 초신성이 터지기 전에 에너지 코어를 모으고 유령을 피하세요.

**단일 HTML 파일**로 만들어진 웹 게임입니다 (HTML5 Canvas 2D + WebAudio). 외부 라이브러리·에셋 없음.

> 원래 한식 컨셉(*부엌 귀신 / Kitchen Gwisin*)에서 **우주 테마로 리스킨**한 버전입니다.
> 게임 규칙·조작·밸런스는 그대로이고, 시각·연출·용어만 우주로 바뀌었습니다.

## ▶ 실행 방법

`index.html` 를 브라우저에서 열면 바로 플레이됩니다. (별도 빌드/설치 불필요)

```bash
# 또는 로컬 서버로
npx http-server . -o
```

## 🎮 조작

| 플랫폼 | 조작 |
|---|---|
| **모바일 (세로)** | 스와이프 4방향 이동 · 💥 **파워** 버튼 · ❄️ **정지** 버튼 · ⏸️ 일시정지 · 🔊 음소거 |
| **데스크톱** | `↑↓←→` / `WASD` 이동 · `Space` 파워 · `F` / `Shift` 정지 · `Esc` / `P` 일시정지 |

## 🕹️ 게임 규칙

- **목표:** 매 스테이지 정해진 개수의 에너지 코어(⚡ 펠릿)를 모으면 → 코어 확보 → 다음 스테이지(더 어려움).
- **플라즈마 파워(💥):** 맵의 플라즈마 코어를 주우면 슬롯(1칸)에 저장. 파워 버튼/`Space` 로 발동하면
  모든 유령이 겁먹어(파랗게) 도망치고, 이때 잡아먹으면 점수(200→400→800→1600). — *공격형*
- **정지장(❄️):** 맵에 나온 정지장 크리스탈을 주워 저장. 정지 버튼/`F` 로 발동하면 **모든 유령이 2.6초간 그 자리에 묶여** 멈춥니다(잡지는 못함). 위기 탈출용 *방어형* 카드로, 플라즈마 파워와 전혀 다른 쓰임새. (스테이지 2+)
- **유령 3종:** ⚪ 유령 드론(무작위 배회) · 🔴 헌터(A\* 추격, 가까우면 가속) · 👾 **유령 우주선**(스테이지 3+, 플레이어 *진행방향 앞*을 노려 길목을 막는 매복형 — 보라빛 촉수 유령).
- **웜홀 순간이동(🌀):** 미로 양 끝의 웜홀 한 쌍은 서로 연결된 포털. 플레이어도, 유령도 통과하면 반대쪽으로 순간이동 → 추격 따돌리기·지름길 심리전. (스테이지 2+)
- **보급품(🛰️):** 일정 시간마다 보급 위성·운석·별똥별 같은 보너스가 미로를 **떠돌아다니다** 사라집니다. 잡으면 큰 점수(200+스테이지×80) **+ 시간 보너스(+4초)**. 움직이는 표적이라 쫓는 재미가 있음.
- **초신성 임박(💥):** 남은 시간 20초 이하가 되면 화면이 백청색으로 물들고 초신성이 떠오릅니다(긴박감 연출).
- **피격:** 목숨 −1, 플레이어·유령 모두 시작 위치로 리셋(연쇄 죽음 방지), 1.5초 무적, **모은 코어는 유지**.
- **패배:** 목숨(3) 소진 또는 시간 초과(초신성 폭발). **런마다 완전 리셋**(로그라이크).
- **점수:** 코어 10 + 콤보(≤×2.0) · 스테이지 확보 500 + 남은 시간 보너스 · **완벽한 확보**(그 스테이지에서 안 맞고 클리어) +250 · 남은 목숨 ×1000(런 종료 시). 최고 점수는 로컬 저장.
- **스테이지 스케일링:** 미로가 세로로 커지고, 유령 수·속도 증가. 5스테이지 이후 무한 진행(하이스코어 도전).

## 🏗️ 기술

- **렌더:** Canvas 2D, 세로 9:16 고정, 화면 크기에 맞춰 셀 크기 자동 계산. 심우주 배경 + 스테이지별 반짝이는 별밭 + 네온 통로 벽.
- **연출(FX):** 몰입감을 위한 화면 흔들림 · 충격파 링 · 임팩트 플래시 · 함선 추진 잔상 · **포커스 비네트**(가장자리를 어둡게 해 시선 집중 · 유령 접근 시 붉게, 파워 시 마젠타) · 히트스톱(피격·포식 순간 정지). 접근성을 위해 `prefers-reduced-motion` 을 존중해 자동 완화됩니다.
- **이동:** 그리드 락 + 입력 버퍼(팩맨식 자동 코너링).
- **미로:** Recursive Backtracking + 벽 8~15% 추가로 뚫어 순환로 생성.
- **AI:** A\*(맨해튼 휴리스틱, 0.4초 리플랜) · 직접 추격(헌터) · **진행방향 앞 매복**(유령 우주선, Pinky 계열) · 랜덤 워크(드론) · Frightened 도주.
- **사운드:** WebAudio 합성 효과음 + 펜타토닉 BGM(잔여 20초 시 템포↑) · 음소거 지원.
- **저장:** localStorage (최고 점수).

### 디버그
`index.html?debug=1` 로 열면 `window.KG` 디버그 훅이 활성화됩니다(수집·파워·정지·보급품·시간 조작 등 테스트용). 평상시에는 비활성.

## ✨ 창작 확장 (오리지널 요소)

팩맨 장르의 손맛은 살리되 "그대로 베낀 게임"으로 느껴지지 않도록 우주 세계관에 녹아드는 오리지널 요소를 더했습니다.

1. **유령 우주선(매복형 AI)** — "각 유령이 성격을 가진다"는 팩맨의 핵심을, 앞길을 가로막는 보라빛 촉수 유령으로 재해석.
2. **웜홀 순간이동** — 팩맨의 좌우 터널을 서로 연결된 웜홀 포털 한 쌍으로 변주(플레이어·유령 공용).
3. **정지장 크리스탈** — 유령을 잠시 묶는 *방어형* 두 번째 카드(플라즈마 파워는 공격형).
4. **움직이는 보급품** — 팩맨의 정지형 과일 대신 *움직이는* 보너스(점수+시간)로 쫓는 재미 추가.
5. **초신성 연출** — "초신성이 터지기 전에" 콘셉트를 종료 20초 전 백청색 섬광·초신성으로 시각화.

**라운드 2 — 동적 액션/전략 요소 (버튼 추가 없이 자동 발동)**

6. **자기장 흡입** — 플라즈마 파워 중 근처 에너지 셀이 함선으로 빨려옵니다. 파워를 셀이 밀집한 곳에서 켜면 "먹이 사냥" 쾌감(타이밍 보상).
7. **유성(☄️)** — 주기적으로 한 줄(행/열)을 관통하는 유성이 등장합니다. 경고선이 뜬 뒤 돌진하니 그 줄을 벗어나세요. 못 피하면 잠시 경직(짧은 무적 부여). 유령 외의 *회피* 위협. (스테이지 2+)
8. **니어미스** — 유령을 아슬아슬하게 스치면(부딪히기 직전) 짧은 슬로모 + 보너스 점수("니어미스!"). 대담한 플레이를 보상.
9. **웜홀 슬링샷** — 웜홀을 통과하면 잠시 가속 + 짧은 무적. 웜홀이 강력한 탈출·재배치 도구가 됩니다.
