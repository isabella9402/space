# ⚡🦊 네온 프라울 (NEON PROWL)

> 네온 미로 속 여우가 되어 에너지 셀을 모으고 적을 따돌리는 **아케이드 미로 게임**.
> 팩맨 장르의 손맛(미로·추격·파워업·콤보)만 취하고 표현은 전혀 다르게 재해석한
> **IP 안전 설계** + **5가지 창작 트위스트**.

**단일 HTML 파일**로 만들어진 웹 게임입니다 (HTML5 Canvas 2D + WebAudio). 외부 라이브러리·에셋 없음(벡터 + 이모지 렌더). **UI는 100% 한국어.**

## ▶ 실행 방법

`index.html` 을 브라우저에서 열면 바로 플레이됩니다. (별도 빌드/설치 불필요)

```bash
# 또는 로컬 서버로
npx http-server . -o
```

## 🎮 조작

| 플랫폼 | 조작 |
|---|---|
| **모바일 (세로)** | 스와이프 4방향 이동 · ⏸️ 일시정지 · 🔊 음소거 |
| **데스크톱** | `↑↓←→` / `WASD` 이동 · `Esc` / `P` 일시정지 · `M` 음소거 |

## 🕹️ 게임 규칙

- **목표:** 미로의 **⚡ 에너지 셀**을 전부 모으면 레벨 클리어 → 다음 레벨(더 어려움).
- **피격:** 적과 닿으면 목숨 −1, 플레이어·적이 시작 위치로 리셋, 1.5초 무적. 목숨(3) 소진 시 게임 오버.
- **점수:** 셀 기본 10점 × 콤보 배수 (러시아워 중엔 ×3). 잔상 스턴 +50, 골든 에그 +500 등.
- **저장:** localStorage 에 최고 점수 · 최고 레벨 · 무한 모드 해금 여부 저장.

## ✨ 5가지 창작 트위스트 (팩맨과의 핵심 차별점 · IP 안전)

1. **에코 트레일** — 🧊 아이스 큐브를 먹으면 8초간 여우 뒤로 **잔상 5개**가 남습니다. 적이 잔상에 닿으면 **3초 스턴 + 50점**. *적을 '먹는' 게 아니라 '함정'에 걸어 묶는다* — 이 게임의 핵심 차별 포인트.
2. **러시아워** — 35초마다 5초간 발동. 모든 에너지 셀이 두 배로 발광하고 **점수 ×3**, BGM 템포도 가속됩니다. 빨간 배너로 카운트다운.
3. **시야 기반 AI** — 적은 **Patrol / Alert** 2상태. 플레이어가 **시야(벽에 가로막히는 레이캐스트)** 안에 들어오면 추격(Alert), 놓치면 3초 뒤 순찰로 복귀. 코너 뒤에 숨거나 적을 유인하는 스텔스 플레이가 가능합니다.
4. **콤보 체인** — 0.6초 안에 셀을 연속 수집하면 배수가 상승 (x1 → x1.5 → x2 → x3 → **x5**). 최적 동선을 짜는 스피드런 보상. (와일드카드 버프 시 상한 x10)
5. **미스터리 에그** — 셀 50%(레벨 5는 30%) 수집 시 등장. 먹으면 5가지 무작위 효과 중 하나:
   - 💰 골든(30%) +500점 · ⚡ 블리츠(25%) 속도 2배 5초 · 💣 봄(20%) 주변 적 스턴 · 🌀 셔플(15%) 적 전원 귀환 · 🌈 와일드카드(10%) 콤보 상한 x10 (15초)

## 👾 적 4종 (각기 다른 행동 · 팩맨 4-모드와 무관)

| 적 | 순찰(Patrol) | 경계(Alert) |
|---|---|---|
| 🤖 **스위퍼** | 우수법으로 벽을 따라 순찰 | A\* 추격 (속도 +20%) |
| 👁️ **헌터** | 제자리에서 90°씩 회전하며 **시야 원뿔**로 감시 | A\* 추격 (속도 +30%) |
| 👻 **고스트** | **벽을 통과**하며 무작위 배회 | 미로를 무시하고 직선 돌진 (느리지만 성가심) |
| 🎭 **미믹** | 에너지 셀로 **위장**해 정지 | 근접(2칸) 시 정체를 드러내고(점프 스케어) 추격 (속도 +25%) |

## 🗺️ 레벨 진행

- **레벨 1~5** (네온 골목 → 베이퍼 그리드 → 사이버 홀 → 회로 미로 → 네온 코어): 미로가 커지고 적 종류·수가 늘어납니다.
- **무한 모드**: 레벨 5 클리어 시 해금. 이후 무작위 레벨이 무한 생성되며, 적 속도가 레벨마다 상승(상한 있음), 미로 크기·적 수가 점점 증가합니다. 사망 시 "최고 레벨"을 기록.
- 적 속도 스케일: `min(1.5, 0.85 + 0.05 × 레벨)` — 아주 느리게 시작해 점점 빨라집니다. 플레이어 속도는 고정.

## 🏗️ 기술

- **렌더:** Canvas 2D, 세로 9:16 고정, 화면 크기에 맞춰 셀 크기 자동 계산. 2겹 네온 벽(핫핑크 테두리 + 발광), 반짝이는 별밭 배경.
- **연출(FX):** 화면 흔들림 · 충격파 링 · 임팩트 플래시 · 히트스톱 · 여우 추진 잔상 · 위험/에코/러시 상태에 따라 색이 바뀌는 **포커스 비네트**. 여우 주스(밥 · 방향전환 스쿼시&스트레치 · 틸트). 접근성을 위해 `prefers-reduced-motion` 존중.
- **이동:** 그리드 락 + 입력 버퍼(자동 코너링). 고스트만 벽 통과 자유 이동.
- **미로:** Recursive Backtracking + 벽 8~15% 추가로 뚫어 순환로 생성.
- **AI:** A\*(맨해튼 휴리스틱) 추격 · 시야 레이캐스트 · 우수법 순찰 · 헌터 시야 원뿔.
- **사운드:** WebAudio 합성 효과음 + C장조 펜타토닉 BGM(러시아워 시 템포↑ · 하이 아르페지오). 음소거 지원.
- **저장:** localStorage (`neon_save`).

### 디버그
`index.html?debug=1` 로 열면 `window.NP` 디버그 훅이 활성화됩니다(레벨 이동·에코·러시·에그·와일드카드·즉시 클리어/피격 등 테스트용). 평상시에는 비활성.

## 📦 범위

레벨 1~5 + 무한 모드, 5가지 트위스트, 적 4종을 갖춘 playable MVP 입니다. 외부 스프라이트 에셋 대신 벡터/이모지로 렌더해 단일 파일로 동작합니다.
