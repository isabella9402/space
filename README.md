# 🍲 부엌 귀신 (Kitchen Gwisin)

> Pac-Man 을 닮은 **한식 로그라이크 미로 수집 게임**. 귀신이 깃든 한옥 부엌에서
> 동트기 전에 재료를 모아 완벽한 한식을 완성하고 귀신을 피하세요.

**단일 HTML 파일**로 만들어진 웹 게임입니다 (HTML5 Canvas 2D + WebAudio). 외부 라이브러리·에셋 없음.

## ▶ 실행 방법

`index.html` 를 브라우저에서 열면 바로 플레이됩니다. (별도 빌드/설치 불필요)

```bash
# 또는 로컬 서버로
npx http-server . -o
```

## 🎮 조작

| 플랫폼 | 조작 |
|---|---|
| **모바일 (세로)** | 스와이프 4방향 이동 · 🌶️ **파워** 버튼 · 🧂 **소금** 버튼 · ⏸️ 일시정지 · 🔊 음소거 |
| **데스크톱** | `↑↓←→` / `WASD` 이동 · `Space` 파워 · `F` / `Shift` 소금 · `Esc` / `P` 일시정지 |

## 🕹️ 게임 규칙 (MVP · M1+M2)

- **목표:** 매 레벨 정해진 개수의 재료(🥬 펠릿)를 모으면 → 요리 완성 → 다음 레벨(더 어려움).
- **김치 파워(🌶️):** 맵의 김치 파워 펠릿을 주우면 슬롯(1칸)에 저장. 파워 버튼/`Space` 로 발동하면
  모든 귀신이 겁먹어(파랗게) 도망치고, 이때 잡아먹으면 점수(200→400→800→1600). — *공격형*
- **소금 부적(🧂):** 맵에 나온 소금을 주워 슬롯에 저장. 소금 버튼/`F` 로 뿌리면 **모든 귀신이 2.6초간 그 자리에 묶여** 멈춥니다(먹지는 못함). 위기 탈출용 *방어형* 카드로, 김치 파워와 전혀 다른 쓰임새. (레벨 2+)
- **귀신 3종:** ⚪ 흰 귀신(무작위 배회) · 🔴 빨간 귀신(A\* 추격, 가까우면 가속) · 👻 **처녀귀신**(레벨 3+, 플레이어 *진행방향 앞*을 노려 길목을 막는 매복형 — 긴 머리 소복 귀신).
- **아궁이 순간이동(🔥):** 미로 양 끝의 부뚜막 화덕 한 쌍은 서로 연결된 포털. 플레이어도, 귀신도 통과하면 반대쪽으로 순간이동 → 추격 따돌리기·지름길 심리전. (레벨 2+)
- **황금 간식(🥮):** 일정 시간마다 약과·경단·만두 같은 황금 간식이 미로를 **떠돌아다니다** 사라집니다. 잡으면 큰 점수(200+레벨×80) **+ 시간 보너스(+4초)**. 움직이는 표적이라 쫓는 재미가 있음.
- **동트기(🌅):** 남은 시간 20초 이하가 되면 하늘이 붉게 물들고 해가 떠오릅니다(긴박감 연출).
- **피격:** 목숨 −1, 플레이어·귀신 모두 시작 위치로 리셋(연쇄 죽음 방지), 1.5초 무적, **모은 재료는 유지**.
- **패배:** 목숨(3) 소진 또는 시간 초과. **런마다 완전 리셋**(로그라이크).
- **점수:** 재료 10 + 콤보(≤×2.0) · 요리 완성 500 + 남은 시간 보너스 · **완벽한 요리**(그 레벨에서 안 맞고 클리어) +250 · 남은 목숨 ×1000(런 종료 시). 최고 점수는 로컬 저장.
- **레벨 스케일링:** 미로가 세로로 커지고(GDD 4.1 표), 귀신 수·속도 증가. 5레벨 이후 무한 진행(하이스코어 도전).

## 🏗️ 기술

- **렌더:** Canvas 2D, 세로 9:16 고정, 화면 크기에 맞춰 셀 크기 자동 계산.
- **이동:** 그리드 락 + 입력 버퍼(팩맨식 자동 코너링).
- **미로:** Recursive Backtracking + 벽 8~15% 추가로 뚫어 순환로 생성.
- **AI:** A\*(맨해튼 휴리스틱, 0.4초 리플랜) · 직접 추격(빨강) · **진행방향 앞 매복**(처녀귀신, Pinky 계열) · 랜덤 워크(흰) · Frightened 도주.
- **사운드:** WebAudio 합성 효과음 + 펜타토닉 BGM(잔여 20초 시 템포↑) · 음소거 지원.
- **저장:** localStorage (최고 점수).

## ✨ 창작 확장 (v1.1)

팩맨 장르의 손맛은 살리되 "그대로 베낀 게임"으로 느껴지지 않도록 한식·귀신 세계관에 녹아드는 오리지널 요소를 더했습니다.

1. **처녀귀신(매복형 AI)** — "각 귀신이 성격을 가진다"는 팩맨의 핵심을, 앞길을 가로막는 긴 머리 소복 귀신으로 재해석.
2. **아궁이 순간이동** — 팩맨의 좌우 터널을 부뚜막 화덕 한 쌍의 포털로 변주(플레이어·귀신 공용).
3. **소금 부적** — "귀신에게 소금 뿌리기" 민속을 살린 *방어형* 두 번째 카드(김치 파워는 공격형).
4. **황금 간식** — 팩맨의 정지형 과일 대신 *움직이는* 보너스(점수+시간)로 쫓는 재미 추가.
5. **동트기 연출** — "동트기 전 완성" 콘셉트를 종료 20초 전 새벽 하늘빛·일출로 시각화.
