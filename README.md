# 🍲 부엌 귀신 (Kitchen Gwisin)

> Pac-Man 을 닮은 **한식 로그라이크 미로 수집 게임**. 귀신이 깃든 한옥 부엌에서
> 동트기 전에 재료를 모아 완벽한 한식을 완성하고 귀신을 피하세요.

**단일 HTML 파일**로 만들어진 웹 게임입니다 (HTML5 Canvas 2D + WebAudio). 외부 라이브러리·에셋 없음.

## ▶ 실행 방법

`index.html` 를 브라우저에서 열면 바로 플레이됩니다. (별도 빌드/설치 불필요)

```bash
# 또는 로컬 서버로
npx http-server . -o
```

## 🕹️ 게임 규칙 (MVP · M1+M2)

- **목표:** 매 레벨 정해진 개수의 재료(🥬 펠릿)를 모으면 → 요리 완성 → 다음 레벨(더 어려움).
- **김치 파워(🌶️):** 맵의 김치 파워 펠릿을 주우면 슬롯(1칸)에 저장. 파워 버튼/`Space` 로 발동하면
  모든 귀신이 겁먹어(파랗게) 도망치고, 이때 잡아먹으면 점수(200→400→800→1600).
- **귀신 3종:** ⚪ 흰(무작위 배회) · 🔴 빨강 Blinky(A\* 로 현재 칸 직행) · 🩷 분홍 Pinky(플레이어가 향한 **4칸 앞**을 노려 길목 차단, L3부터 등장).
- **피격:** 목숨 −1, 플레이어·귀신 모두 시작 위치로 리셋(연쇄 죽음 방지), 1.5초 무적, **모은 재료는 유지**.
- **패배:** 목숨(3) 소진 또는 시간 초과. **런마다 완전 리셋**(로그라이크).
- **점수:** 재료 10 + 콤보(≤×2.0) · 요리 완성 500 + 남은 시간 보너스 · **완벽한 요리**(그 레벨에서 안 맞고 클리어) +250 · 남은 목숨 ×1000(런 종료 시). 최고 점수는 로컬 저장.
- **레벨 스케일링:** 미로가 세로로 커지고(GDD 4.1 표), 귀신 수·속도 증가. 귀신 속도는 초반엔 확실히 느리고 스테이지마다 **완만하게** 빨라져요(L1 ≈ 0.56×플레이어 → 상한 1.15×). 5레벨 이후 무한 진행(하이스코어 도전).
- **게임 방법:** 메인 메뉴의 "게임 방법 ❓" 버튼(첫 실행 시 자동 표시)에서 한국어 튜토리얼 확인.

### 🎯 팩맨다운 추가 요소
- **🌀 워프 터널:** 좌우 옆문으로 순간이동. 귀신은 터널 안에서 느려져(고전 팩맨) 도주로가 됨.
- **🍜 보너스 요리:** 재료 40%/75% 지점에 한국 길거리 음식(만두·떡꼬치…)이 ~9초 등장 → 큰 점수(팩맨의 "과일").
- **🔴 Scatter/Chase 리듬:** 빨간 귀신이 추격(12초)↔산개(구석으로, ~5초)를 반복하고 전환 시 방향을 반전. 산개 중엔 옅은 색으로 표시(압박↔휴식 리듬).
- **❤️ 1UP:** 8,000점마다 목숨 +1(최대 5).

## 🏗️ 기술

- **렌더:** Canvas 2D, 세로 9:16 고정, 화면 크기에 맞춰 셀 크기 자동 계산. 캐릭터·펠릿·유령은 **귀여운 카와이 벡터**(큰 눈·볼터치 고추 요리사, 몽글 유령, 당고·모찌, 표정 있는 고추, 반짝 워프 포탈, 별 파티클). (외부 이미지 없이 전부 canvas로 그림.)
- **캔디 벽 (레벨마다 변경):** 벽은 **또렷한 픽셀 아트 캔디 타일**(하드 픽셀 블록 + 픽셀 베벨/외곽선/장식, nearest-neighbor 확대). 레벨마다 테마가 바뀜 — 딸기(둥근·하트)·민트(도트)·초코(각진·크림 줄무늬)·블루베리(별)·레몬·포도·솜사탕·말차(각진·초코칩) 8종 순환. 색·모서리 모양(둥근/각진)·바닥색·장식 무늬가 테마마다 다름.
- **플레이어:** 정통 **팩맨**(입이 열린 쐐기로 벌어지며 진행 방향으로 회전) + 귀여운 눈·볼터치·반짝 트레일.
- **이동:** 그리드 락 + 입력 버퍼(팩맨식 자동 코너링).
- **미로:** Recursive Backtracking + 벽 8~15% 추가로 뚫어 순환로 생성.
- **AI:** A\*(맨해튼 휴리스틱, 0.4초 리플랜) · 랜덤 워크 · Frightened 도주.
- **사운드:** WebAudio 합성 효과음 + 펜타토닉 BGM(잔여 20초 시 템포↑) · 음소거 지원.
- **저장:** localStorage (최고 점수).
